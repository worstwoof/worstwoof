# 🚀 我引以为豪的项目 (My Proudest Projects) 🚀

</div>

### 1. ✨ “掌间流明” (Palm Lumen)
* 一个结合 **YOLOv5**、**OpenCV** 和 **Qt** 的酷炫应用！
* 它能实时检测手掌位置，并在你的手心“凭空”放出**烟花粒子特效**。
* 我把效果发到B站，一周内获得了 **1.5万+** 的播放量！

### 2. 🧠 复现 RetinaNet (Focal Loss)
* **独立开发者**项目，深入研究 AI 论文。
* 使用 **Jittor** 框架，从零开始完整复现了《Focal Loss for Dense Object Detection》论文。
* 涵盖了从数据加载、FPN、Focal Loss 到模型训练的全部核心流程。

### 3. 🎸 SpaceMusic 音乐网站
* 作为**团队成员**，使用 **HTML/CSS/JavaScript** 开发。
* 我主要负责了核心的**播放器组件**和 **MV 详情页面**的制作。

### 4. 🏭 卓越工程师训练营
* 深入真实的工业界，使用 **Halcon** 视觉软件。
* 独立完成了针对**芯片表面缺陷检测**和尺寸测量的项目。
